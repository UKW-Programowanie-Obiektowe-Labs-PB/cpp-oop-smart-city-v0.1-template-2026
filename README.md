# Smart City IoT — Projekt Laboratorium (v0.1)

Minimalistyczny szablon startowy dla projektu "Smart City IoT" w C++20.

## Quick Start

1. Sklonuj: git clone [URL]
2. Wejdź do folderu: cd [NAZWA]
3. Buduj: make build
4. Uruchom: ./bin/app (lub make run)

## Zasady

- Zakaz edycji kodu źródłowego przez interfejs WWW GitHub.
- Ostrzeżenia kompilatora (Warnings) traktujemy jako błędy.

## Public PC Safety

Po zajęciach na komputerach publicznych usuń globalne ustawienia: `git config --global --unset`.
